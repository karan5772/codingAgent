## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Usage

**Set up your `.env` file:**

```
OPENAI_API_KEY=your_openai_api_key_here
```

**Run the agent with:**

```bash
node index.js
```
